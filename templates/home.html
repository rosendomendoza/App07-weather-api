Wheather Data API

For a Station in a particular day:

URL format: http://127.0.0.1:5000/api/v1/station/date

Example : http://127.0.0.1:5000/api/v1/10/2000-10-24


For all data of a particular station:

URL format: http://127.0.0.1:5000/api/v1/station

Example : http://127.0.0.1:5000/api/v1/10


For data of a particular station in range of date:

URL format: http://127.0.0.1:5000/api/v1/station/from/to

Example : http://127.0.0.1:5000/api/v1/10/1900-01-01/1950-12-31
