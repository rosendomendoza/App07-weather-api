Europe Weather Forecast API

Klein Tank, A.M.G. and Coauthors, 2002. Daily dataset of 20th-century surface air temperature and precipitation series for the European Climate Assessment. Int. J. of Climatol., 22, 1441-1453. Data and metadata available at https://www.ecad.eu


For all data of a particular station:

URL format: http://127.0.0.1:5000/api/v1/all/station

Example : http://127.0.0.1:5000/api/v1/all/10


For a Station in a particular Year:

URL format: http://127.0.0.1:5000/api/v1/yearly/station/year

Example : http://127.0.0.1:5000/api/v1/yearly/10/2000


For a Station in a particular day:

URL format: http://127.0.0.1:5000/api/v1/day/station/date

Example : http://127.0.0.1:5000/api/v1/day/10/2000-10-24


For data of a particular station in range of date:

URL format: http://127.0.0.1:5000/api/v1/range/station/from/to

Example : http://127.0.0.1:5000/api/v1/range/10/1900-01-01/1950-12-31
